## 작업 목록 (우선순위 순)

- **목적**: 팀 내부 공유용 문서

- **적용 단계**: L1 전처리

### 3. 대화 생성
- **목적**: 대화 씬에서 대사 스크립트 자동 생성
- **적용 단계**: L2
- **상세 스펙**: [specs/dialogue.md](./dialogue.md)

### 4. Technique DB 구축
- **목적**: 유튜브 영상 분석 → 연출 테크닉 DB 축적

- **상세 스펙**: [specs/technique_db.md](./technique_db.md)

## 레벨별 역할 재정의

인터뷰 결과, 각 레벨의 역할이 명확해짐:

| Level | 역할 | 담당 영역 |
|-------|------|----------|
| **L1** | Scene Architect | 스토리 → 씬 분할 + **서사 확장 (펌프업)** |
| **L2** | Shot Composer | 씬 → 샷 시퀀스 + **대사 생성** + 샷 메타데이터 |
| **L3** | Prompt Builder | 샷 → 최종 프롬프트 + **연출 테크닉 DB** |

### L2 vs L3 역할 분리

| 영역 | L2 (Shot Composer) | L3 (Prompt Builder) |
|------|-------------------|---------------------|
| **샷 구조** | 샷 타입, 길이, 순서 | - |
| **스토리 요소** | 대사 유무, 액션 유형, 감정 태그 | - |
| **샷 관계** | 전후 샷 연결, 장르 분류 | - |
| **연출 테크닉** | - | 카메라 워크, 조명, 효과 |
| **시각 디테일** | - | 배우 피부, 공기 먼지, 렌즈 특성 |
